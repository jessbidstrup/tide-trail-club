 <h1>Tide & Trail Club - Responsive Design Challenge Project</h1>
    <span class="tag">HTML</span>
    <span class="tag">CSS</span>
    <span class="tag">Responsive Design</span>
    <span class="tag">Flexbox</span>
    <h2>Overview</h2>
    <p>
        Tide & Trail Club is a fictional coastal outdoor club created as a front-end
        challenge project. The goal of this project was to design and build a clean,
        responsive website using semantic HTML and modern CSS techniques.
    </p>
    <p>
        The site focuses on clear content structure, accessibility, and a calm,
        nature-inspired visual identity suitable for a real-world community club.
    </p>
    <h2>Project Goals</h2>
    <ul>
        <li>Create a realistic, portfolio-ready club website</li>
        <li>Practice semantic HTML structure</li>
        <li>Use Flexbox for layout and responsiveness</li>
        <li>Apply a consistent colour and typography system</li>
        <li>Design for both desktop and mobile screens</li>
    </ul>
    <h2>Features</h2>
    <ul>
        <li>Fixed navigation bar</li>
        <li>Full-height hero section with background image</li>
        <li>Split-layout content sections</li>
        <li>Responsive activity and values cards</li>
        <li>Hover states and visual feedback</li>
        <li>Mobile-friendly layout using media queries</li>
    </ul>
